# 国际化

Nami UI 使用 `@lit/localize` 的 runtime mode。活动语言可以在不刷新页面的情况下切换，本地化模块准备完成后，相关 Lit 组件会重新渲染。

## 运行时边界

`nami-config` 是全局 locale 控制器。它独立于 `nami-theme`，所以语言配置和视觉主题不会混在一起。

## 规则

| 规则 | 原因 |
| --- | --- |
| 组件内部 fallback 文案使用 `msg()` | 方便 `@lit/localize` 提取和替换 |
| 动态纯文本使用 `msg(str\`...\`)` | 保留翻译占位符 |
| 长教程使用 locale Markdown 文件 | 文档更可读，也更容易编辑 |
| 不拼接可翻译句子 | 不同语言的语序不同 |
| 日期和数字使用 `Intl` | 格式化规则与 locale 有关 |
