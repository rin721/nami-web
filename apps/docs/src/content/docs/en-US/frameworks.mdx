# Frameworks

Nami UI does not require framework wrappers for the first release. Use standard custom element registration and pass data through attributes, properties, and DOM events.

## Native HTML

```html
<script type="module">
  import '@nami/themes/default.css';
  import '@nami/ui/register';
</script>

<nami-button>Save</nami-button>
```

## Vue

```ts
import '@nami/themes/default.css';
import '@nami/ui/register';
```

Configure Vue to treat `nami-*` tags as custom elements in the application compiler options.

## React

```tsx
import '@nami/themes/default.css';
import '@nami/ui/register';

export function App() {
  return <nami-button>Save</nami-button>;
}
```

For typed React events, attach listeners through refs or bridge events in a small local wrapper.
