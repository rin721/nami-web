# i18n

Rin UI uses `@lit/localize` in runtime mode. The active locale can change without a page reload, and localized Lit components re-render when the locale module is ready.

## Runtime boundary

`rl-config` is a global locale controller. It is separate from `rl-theme`, so language and visual theme remain independent.

## Rules

| Rule | Why |
| --- | --- |
| Use `msg()` for component fallback text | It lets `@lit/localize` extract and replace strings |
| Use `msg(str\`...\`)` for dynamic plain text | It preserves placeholders for translation |
| Keep long tutorials in locale Markdown files | Docs remain readable and easy to edit |
| Do not concatenate translatable sentences | Word order changes by language |
| Use `Intl` for dates and numbers | Formatting is locale specific |
