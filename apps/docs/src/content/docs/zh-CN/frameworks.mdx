# 框架接入

Nami UI 第一版不需要框架 wrapper。使用标准自定义元素注册，并通过 attributes、properties 和 DOM events 传递数据。

## 原生 HTML

```html
<script type="module">
  import '@nami/themes/default.css';
  import '@nami/ui/register';
</script>

<nami-button>保存</nami-button>
```

## Vue

```ts
import '@nami/themes/default.css';
import '@nami/ui/register';
```

在 Vue compiler options 中将 `nami-*` 标签配置为 custom elements。

## React

```tsx
import '@nami/themes/default.css';
import '@nami/ui/register';

export function App() {
  return <nami-button>Save</nami-button>;
}
```

需要类型化 React 事件时，可以通过 refs 监听，或在项目内写一个很小的事件桥接 wrapper。
