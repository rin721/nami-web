# 主题引擎

`nami-theme` 是视觉主题边界。它控制嵌套组件的 seed accent、主题模式、密度、动效和 style preset。

## 这里放什么

Theme 集中说明主题边界、tokens 分层、派生算法、风格 preset、Theme Designer 和 Theme Lab。这里讨论如何生成、覆盖、导出和验证主题。

## 不放什么

组件 API 不放在 Theme；每个组件可覆盖的 parts、states 和 component tokens 在组件详情页维护。官网外观偏好也不放在 Theme，它们属于 Settings 工具区。

## 下一步去哪

先读 Tokens 理解分层，再读主题算法理解派生过程；需要直接调色时进入 Theme Designer，需要做组合验证时进入 Theme Lab。

## Token 层级

| 层级 | 示例 | 作用 |
| --- | --- | --- |
| Seed | `--nami-accent-50`, `--nami-motion-normal` | 品牌和动效输入 |
| Semantic | `--nami-color-primary`, `--nami-surface`, `--nami-text` | 跨组件共享语义 |
| Component | `--nami-button-bg`, `--nami-chip-selected-bg` | 单组件覆盖边界 |

hover、selected、focus、switch checked 和 loading indicator 都应该跟随同一个 accent seed。
