# 风格 Preset

组件源码保持不变，视觉皮肤通过 tokens 切换。默认 preset 是中性的社区组件风格。插画 preset 增加粗描边、偏移阴影、纸张表面和手绘背景纹理。

## Preset API

```html
<nami-theme style-preset="default"></nami-theme>
<nami-theme style-preset="illustration"></nami-theme>
<nami-theme style-preset="ant-illustration"></nami-theme>
```

`ant-illustration` 保留为兼容别名，公开行为等同于 `illustration`。
