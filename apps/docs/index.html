Quick Start

Use standard elements, not framework wrappers

Register every element for prototypes, or import one component entry when a production app needs a smaller dependency surface.

Full register 
import '@rin-labs/themes/default.css';
import '@rin-labs/ui/register';
Single component 
import '@rin-labs/themes/default.css';
import '@rin-labs/ui/button';
HTML surface 
<rl-button>Save</rl-button>
<rl-input label="Search"></rl-input>

Frameworks

One package across HTML, Vue, and React

Rin UI ships standards-based custom elements. Framework integration is mostly registration, compiler config, and event handling.

Native HTML 
<script type="module">
  import '@rin-labs/ui/register';
</script>
Vue 
compilerOptions: {
  isCustomElement: tag => tag.startsWith('rl-')
}
React 
useEffect(() => {
  ref.current?.addEventListener('rl-change', onChange);
}, []);
