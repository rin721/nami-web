# Rin Web UI Labs

Lit + TypeScript Web Components UI library with a fixed deep interaction system and replaceable visual skins.

## Packages

- `@rin-labs/ui`: core `rl-*` web components.
- `@rin-labs/themes`: default theme CSS and theme helpers.
- `@rin-labs/tokens`: token names and shared design constants.
- `@rin-labs/playground`: native HTML playground.

## Development

```bash
npm install
npm run build
npm test
npm run dev
```
